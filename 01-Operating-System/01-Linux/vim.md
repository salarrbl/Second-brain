### Hot Command Mode
- :w
	- save changes
	- :w path and name 
		- save as
-  :q
	- exit
- :wq  & :x ZZ
	-  save and exit
- :12
	- go line 12
- set un
	- set number
- :e path and filename
	- open file
-  :q! 
	- not save and exit
- !system command 
- .!
